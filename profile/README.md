# cuioss

Open source Java libraries and tools for enterprise application development.

## Projects

Our main projects include:

- **OAuth Sheriff** - JWT/OAuth token validation library for Quarkus
- **CUI Java Tools** - Common utilities for Java development
- **CUI Test Libraries** - Testing utilities and generators

## Getting Started

Most of our libraries are available on Maven Central:

```xml
<dependency>
    <groupId>de.cuioss</groupId>
    <artifactId>library-name</artifactId>
    <version>latest</version>
</dependency>
```
